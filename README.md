# Admin Dashboard UI

Responsive **Admin Dashboard** built with **HTML, CSS and Vanilla JavaScript**.

This project simulates a real admin panel used in SaaS and e-commerce platforms, focusing on **layout architecture**, **component structure**, and **clean frontend logic** without using frameworks.
